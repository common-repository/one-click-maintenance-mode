A clean and simple plugin to turn on maintenance mode on WordPress sites.

== Description ==
A clean and simple plugin to turn on maintenance mode on WordPress sites.

Simply activate the plugin to turn on maintenance mode and deactivate to turn it off. No configuration needed!

After the maintenance mode is turned on, only the logged-in admin users will be able to access the site.

There's no extra unnecessary code in the plugin, thus you can stay assured that it won't affect your site's performance at all.

== Installation ==
1. Upload the folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the “Plugins” menu in WordPress.
